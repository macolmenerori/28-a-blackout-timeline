# 28-a-blackout-timeline

On April 28, 2025 there was an electricity blackout on the Iberian Peninsula. On autumn 2025 the Spanish Government released an official report which analyzes the causes of the blackout.

This web application is a timeline summarizing the events detailed on the official report.

Check out the application at [https://apagon-28-a.miguelangelcolmenero.es](https://apagon-28-a.miguelangelcolmenero.es)
